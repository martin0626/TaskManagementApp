This is My Task App

Frontend : React
Backend: Django REST Framework


Frontend Start: 

Install: install -g npm

Before Start App, Must Be Installed:

	..\TaskManagementApp\frontend-react: npm install react-router-dom
	

Path: ..\TaskManagementApp\frontend-react: npm start app


Backend Start: 

After you open backend-django folder with Code Editor Navigate To ..\TaskManagementApp\backend-django\task_management

..\TaskManagementApp\backend-django\task_management: pip install requrements.txt


After React App Start: 

..\TaskManagementApp\backend-django\task_management: python manage.py runserver


Finish:

Open Browser At : http://localhost:3000/
